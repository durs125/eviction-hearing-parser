# eviction-hearing-parser
parse registers of actions for eviction hearings
Eviction hearings are found on this website:
https://odysseypa.traviscountytx.gov/JPPublicAccess/Search.aspx?ID=900

Goal is to determine when people are being evicted and provide our partners this information so they can provide the tenants lawyers.

Scraper should pull tenant, plaintiff, last even date, and last event type. These should only be pulled for eviction cases.
